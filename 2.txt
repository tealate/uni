unityにて物体の任意の座標に対して任意の方向から力を加える際、その物体が動くために必要な力を取得するC#のスクリプトを作ってほしい

私が理解するに、あなたは物体に対して任意の力を加えたいと考えていますね。その力が物体をどれだけ動かすかを知りたいということでしょうか？

もしそうならば、以下のようなスクリプトを作ることができます。

Copy code
using UnityEngine;

public class ForceCalculator : MonoBehaviour
{
    // 物体
    public Rigidbody target;

    // 力を加える位置
    public Vector3 forcePosition;

    // 力を加える方向
    public Vector3 forceDirection;

    // 力の強さ
    public float forceMagnitude;

    // 動かすために必要な力を計算します
    public void CalculateRequiredForce()
    {
        // 物体の質量
        float mass = target.mass;

        // 力を加える位置から物体の重心までのベクトル
        Vector3 lever = target.centerOfMass - forcePosition;

        // 力を加える方向に垂直なベクトル
        Vector3 perpendicularDirection = Vector3.Cross(lever, forceDirection);

        // 動かすために必要な力
        Vector3 requiredForce = mass * perpendicularDirection;

        // 力の強さを調整
        requiredForce *= forceMagnitude;

        // 結果を表示
        Debug.Log(requiredForce);
    }
}

このスクリプトを使用するには、次のようにします。

UnityのSceneビューに物体を配置します。
物体にRigidbodyコンポーネントを追加します。
物体に上記のスクリプトをアタッチします。
スクリプトのtargetプロパティに物体のRigidbodyコンポーネントを設定します。
スクリプトのforcePositionプロパティに力を加える位置を設定します。
スクリプトのforceDirectionプロパティに力を加える方向を設定します。
スクリプトのforceMagnitudeプロパティに力の強さを設定します。
